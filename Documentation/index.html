HTML Structure

Sonorama template is based on a responsive structure that look beautiful at any size, be it a 17" laptop screen or an iPhone.

It has the flexibility to go mobile like a champ. You can really easily customize all part of site or build new.

CSS Structure

I used some CSS files in this template (all styles're imported in main file of CSS - style.css). By using of them, we can work round this. Those files also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the files below:

  1. Sonorama.css, main.css - main css files;
  2. responsive.css - media query layouts
  3. normalize.css, isotope.css- contains third party styles
  4. color folder contains 8 diferent colors styles and the dark layout.
  5. vegas and animate folders contains third party styles

If you create you own styles you must check all the sizes of responsive so all works ok, if you need to change your own styles at certain size you will do it at responsive.css.

Choosing Color

Colors

The default color of this template is red, if you want to change it for any of the eight predefined colors open the file js/plugins.js and replace the color in the following line:

var currentColor = 'red';

You can choose between the following colors:

blue
black
green
yellow
orange
purple
red
turquoise

You also can choose between a light or dark layout, the deafault one is light layout, if you want the dark one just insert the following class at body of your index.html

<body class='darker'>
